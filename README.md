# 🚀 Serverless Tasks App (AWS)

This is a full-stack serverless web application built using AWS services.

## 🌍 Live Demo
http://jolomi-tasks-app.s3-website-eu-west-1.amazonaws.com/

---

## 🛠️ Tech Stack

- AWS API Gateway (REST API)
- AWS Lambda (Python)
- Amazon DynamoDB
- Amazon S3 (Frontend Hosting)

---

## ⚙️ Features

- Create tasks
- View tasks
- Delete tasks
- Serverless backend

---

## 🧠 What I Learned

- Building REST APIs with API Gateway
- Lambda function integration
- Handling CORS issues
- DynamoDB CRUD operations
- Deploying frontend using S3
